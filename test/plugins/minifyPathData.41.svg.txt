Don't modify path if there is an attribute selector on "d".

===

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 20 20">
  <style>
    path {stroke:red;} 
    path[d*=&quot;h 1&quot;] { stroke: green;}
  </style>
  <path d="M 1 1 h 10"/>
</svg>

@@@
